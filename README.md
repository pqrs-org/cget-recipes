# cget-recipes

Recipes for [cget](https://github.com/pfultz2/cget).

## Install

```shell
cget install pqrs-org/cget-recipes
```

## Update recipes

```shell
GITHUB_TOKEN=YOUR_GITHUB_TOKEN make update-package-version
```

Replace YOUR_GITHUB_TOKEN with your read only GitHub token.
